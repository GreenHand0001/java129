# 学生宿舍信息管理系统 java129

## 项目概述

学生宿舍信息管理系统是基于前后端分离架构开发的，旨在帮助学校高效管理宿舍相关事务。系统支持多种角色使用，包括系统管理员、宿舍管理员和学生，提供全面的宿舍管理功能。

## 技术栈

- **开发环境**：
  - IntelliJ IDEA
  - JDK 1.8
  - Maven
  - Node.js

- **技术框架**：
  - Java
  - Spring Boot
  - MyBatis-Plus
  - Vue

- **数据库**：
  - MySQL

- 公告信息管理
- 报修信息管理

#### 宿舍管理员模块
- 学生信息管理

- 报修信息管理

## 运行环境

- 服务器：推荐使用 Linux 服务器
- 数据库：MySQL 5.7 或更高版本
- 前端：现代浏览器（例如：Chrome、Firefox、Edge）
- 后端：JDK 1.8 或更高版本

## 核心亮点

- **前后端分离**：系统采用前后端分离的架构，前端使用 Vue，后端使用 Spring Boot，有效提升了开发效率与系统性能。
- **功能完善**：覆盖宿舍管理的各个方面，满足不同角色的需求。
- **易于维护**：采用模块化设计，便于扩展和维护。
- **高兼容性**：支持多种操作系统和数据库环境，适应不同学校的技术需求。
